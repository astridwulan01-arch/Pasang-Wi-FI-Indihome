# 🌐 Landing Page Jasa Pemasangan Baru WiFi IndiHome

Repository ini berisi kode sumber (*source code*) untuk website satu halaman (Landing Page) yang dirancang khusus sebagai media promosi digital bagi Sales/Mitra Resmi Pemasangan Baru IndiHome. 

Website ini dibuat dengan fokus pada kecepatan performa, desain modern yang responsif, serta optimasi konversi (*Conversion Rate Optimization*) agar calon pelanggan dapat langsung terhubung ke WhatsApp Sales dengan sekali klik.

---

## ✨ Fitur Utama Website

* 📱 **Desain Responsif:** Tampilan yang bersih dan proporsional saat diakses melalui smartphone, tablet, maupun komputer.
* ⚡ **Ringan & Cepat:** Dibangun menggunakan Tailwind CSS (via CDN) tanpa *framework* JavaScript yang berat, memastikan *loading* halaman super cepat.
* 🟢 **Integrasi WhatsApp (CTA Premium):** Tombol aksi (*Call to Action*) yang dinamis, langsung mengarahkan teks pesan spesifik sesuai paket yang dipilih calon pelanggan.
* 📑 **Struktur Konten Teroptimasi:** * **Hero Section:** Penawaran memikat di awal halaman.
  * **Fitur & Keunggulan:** Alasan kuat kenapa harus memilih layanan ini.
  * **Pricelist (Tabel Harga):** Informasi paket internet yang transparan dan rapi.
  * **Alur Pendaftaran:** Edukasi 4 langkah mudah pasang WiFi.
  * **FAQ Section:** Menjawab pertanyaan umum untuk mengurangi keraguan pelanggan.

---

## 🛠️ Teknologi yang Digunakan

* **HTML5** - Struktur semantik halaman web.
* **Tailwind CSS** - Framework CSS modern untuk desain UI yang cepat dan estetik.
* **FontAwesome 6** - Menyediakan ikon-ikon navigasi yang interaktif.

---

## 🚀 Cara Menggunakan & Kustomisasi

1. **Clone atau Download** repository ini.
2. Buka file `index.html`.
3. Cari teks nomor WhatsApp default `628551929659` di dalam kode, lalu ubah dengan nomor WhatsApp Anda sendiri (gunakan format kode negara `62`).
4. Sesuaikan harga paket internet atau promo di bagian `#paket` jika ada perubahan tarif resmi dari Telkom.
5. Upload kembali ke GitHub Pages untuk langsung mengonlinekannya!
